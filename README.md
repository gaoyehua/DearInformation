# DearInformation
this is a usefull Android application,it can help you gain more information but not waste lots of time.
亲亲资讯
===
这是一款融合新闻，知识，美图，精品视频为一体的资讯应用，让人们用最短的时间获取最多的资讯！

应用下载:
  [亲亲资讯](http://fir.im/cykw)
